# 使用说明

## 启动程序

### 方法一：使用批处理文件（推荐）
双击运行 `run.bat` 文件

### 方法二：命令行启动
```bash
python main.py
```

## 功能使用

### 1. 单个元件查询
1. 在"输入元件型号"文本框中输入元件型号
2. 点击"搜索价格"按钮

### 2. 批量元件查询
1. 在"批量输入"文本框中每行输入一个元件型号
2. 点击"搜索价格"按钮

### 3. 文件导入查询
1. 点击"浏览"按钮选择文件
2. 选择Excel文件（.xlsx）或文本文件（.txt）
3. 点击"搜索价格"按钮

### 4. 下载Excel模板
点击"下载Excel模板"按钮，选择保存位置创建模板文件

### 5. 上传填好的模板
点击"上传填好的模板"按钮，选择已填写的Excel文件导入元件型号

### 6. 导出结果
点击"导出结果到Excel"按钮，选择保存位置导出查询结果

## 文件格式说明

### Excel文件格式
- 必须包含"元件型号"列（也支持"Part Number"、"型号"、"元件编号"等列名）
- 可以包含其他列如"元件描述"等（可选）

### TXT文件格式
- 每行一个元件型号
- 空行将被忽略

## API密钥配置

在 `config.py` 文件中配置您的Mouser API密钥：

```python
MOUSER_API_KEYS = [
    "您的API密钥1",
    "您的API密钥2",
    "您的API密钥3"
]
```

## 结果说明

查询结果包含以下字段：
- **元件型号**：用户输入的元件型号
- **搜索型号**：实际在Mouser中搜索到的型号
- **产品名称**：产品名称
- **品牌**：制造商
- **价格(USD)**：最大批次对应的价格
- **最小批次**：价格对应的最小购买数量
- **库存**：当前库存情况
- **备注**：查询状态说明
  - 空白：成功找到精确匹配
  - "相似型号爬取"：找到相似型号
  - "未找到"：未找到任何匹配
  - "错误: ..."：发生错误

## 注意事项

1. 请确保网络连接正常
2. API密钥需要在Mouser官网申请
3. 程序已实现速率限制控制，避免被API限制
4. 当查询大量元件时，建议分批进行以避免超时
5. 如果遇到API限制，请增加 `REQUEST_DELAY` 参数值
